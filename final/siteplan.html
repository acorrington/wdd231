Roseburg Volunteer Network Site Plan

Site Name

Name: Roseburg Volunteer Network

Reason: This name reflects the website's focus on fostering volunteerism in Roseburg, OR, emphasizing community engagement and local impact. It is clear, memorable, and aligns with the site's mission to connect residents with service opportunities. A potential domain like roseburgvolunteers.org would enhance branding, though not required for this project.

Site Purpose

The Roseburg Volunteer Network website serves as a hub to promote volunteer opportunities in Roseburg, OR. It provides information on local events (e.g., Umpqua River cleanups, youth tutoring), displays dynamic event listings from a JSON file or API, and offers a form for residents to sign up as volunteers. The site aims to encourage community participation, highlight local impact, and provide accessible, user-friendly content for all residents.

Scenarios

  • What volunteer opportunities are available in Roseburg this month, and how can I filter them by type (e.g., environmental, educational)?
  • How do I sign up to volunteer, and what information is needed to join the Roseburg Volunteer Network?

Color Schema

The color scheme emphasizes community and trust:

  • Green (#1B5E20): Used for headings (h1, h2, h3) and navigation background to reflect Roseburg's natural environment.
  • Light Blue (#4FC3F7): Applied to buttons, links, and accents to convey friendliness and interactivity.
  • White (#FFFFFF): Background color for main content areas to ensure cleanliness and readability.
  • Dark Gray (#333333): Used for body text and secondary elements for high contrast and accessibility.

Typography

The typography ensures readability and a modern aesthetic:

  • Roboto (Regular, Bold): Used for headings (h1, h2, h3) and body text across the site for consistency and clarity.
  • Open Sans (Regular): Applied to captions, footer links, and secondary text for subtle contrast and visual hierarchy.

Fonts are sourced from Google Fonts for accessibility and web compatibility.

Wireframe

Mobile View (320px)

A single-column layout for the home page (index.html):

  • Header: Hamburger menu icon (top-left) and logo (top-right, text-based “Roseburg Volunteer Network”).
  • Hero Section: Full-width image of volunteers at Umpqua River, with a headline (“Join Roseburg Volunteers!”) and a call-to-action button (“Explore Opportunities”).
  • Featured Event: A single event card with title, date, location, and image, centered.
  • Weather Widget: A small section showing current Roseburg weather (e.g., temperature, condition).
  • Footer: Links to Opportunities, Join Us, Attributions, and Video Demo, stacked vertically.

Desktop View (768px+)

A multi-column layout for the home page:

  • Header: Horizontal navigation bar with links (Home, Opportunities, Join Us) and logo centered.
  • Hero Section: Wide image of volunteers, with headline and button centered over the image.
  • Featured Event: Event card (title, date, location, image) in a two-column layout next to the weather widget.
  • Weather Widget: Displays current temperature, condition, and 3-day forecast.
  • Footer: Horizontal links to Opportunities, Join Us, Attributions, and Video Demo.

Wireframes focus on content placeholders, with responsive design using CSS Grid and media queries.
